# 🎬 Estúdio Infinito Premium > Um ecossistema digital imersivo de alta performance, desenvolvido com tecnologias modernas de última geração para entregar uma experiência premium única. --- ## 🚀 Tecnologias Utilizadas O projeto foi construído utilizando um ecossistema robusto e nativo, focado em performance, tipagem estática e escalabilidade: * **Next.js 15 (App Router):** Framework React para produção com renderização otimizada. * **React 19 & TypeScript:** Interface reativa com tipagem forte e segura de ponta a ponta. * **Tailwind CSS:** Estilização utilitária de alta performance para designs modernos e fluidos. * **Supabase:** Backend-as-a-Service (BaaS) seguro e em tempo real. * **Lucide React:** Pacote de ícones limpos, consistentes e otimizados. --- ## 📂 Estrutura de Pastas do Projeto A arquitetura foi organizada de forma modular para garantir fácil manutenção e escalabilidade: ```text estudio-infinito-premium/ ├── .github/ # Fluxos de automação e templates do GitHub ├── public/ # Arquivos estáticos (imagens, fontes, assets) ├── src/ │ ├── app/ # Rotas, páginas e layouts (Next.js App Router) │ ├── components/ # Componentes visuais reutilizáveis │ ├── hooks/ # Hooks customizados para lógica de estado │ ├── lib/ # Configurações de clientes externos (Supabase, etc.) │ ├── types/ # Definições de tipos globais do TypeScript │ └── utils/ # Funções utilitárias e auxiliares ├── .gitignore # Arquivos ignorados pelo Git ├── package.json # Dependências e scripts do projeto ├── tailwind.config.ts # Customização de temas e design system └── tsconfig.json # Configurações do compilador TypeScript
